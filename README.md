# Meta POP Agent

## 🚀 価格ではなく「体験」で売れるか？
AIエージェントが仮想売場を回遊し、  
**「価格訴求POP」と「体験数字POP」どちらを選ぶか**を可視化する実証プロジェクト

## 🧠 コンセプト

従来の売場：
- 価格（いくら安いか）

これからの売場：
- 体験（どんな価値か）

---

### 💡 メタPOP構想
> 「支払数字」から「体験数字」へ

例：
- 198円 → ❌
- 糖度12度（いちご並み） → ⭕

👉 人は価格ではなく「意味」で動く

---

## 🧪 実証内容

AIエージェントを仮想売場に配置し、購買行動をシミュレーション

### 売場構成
- 左：価格POP（198円）
- 右：体験POP（糖度12）

### エージェント属性
- 価格重視
- 品質重視
- 健康志向
- 家族志向
- 発見好き
- 料理好き

👉 属性ごとに意思決定が変化

---

## 📊 実証結果

- 体験数字POPが多数のエージェントに支持される
- 価格重視層のみ価格側へ

👉 多くの消費者は「意味」で選ぶ
