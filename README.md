# Meta POP Agent

## 🧠 概要

価格ではなく「体験数字」で購買意思決定を支援するAIエージェント。
メタPOP（非価格訴求）を自動生成し、売場の価値伝達を最適化します。

---

## 🚀 デモ

```bash
python main.py
```

出力例：
この商品は糖度12度（イチゴ並み）です

---

## 💡 背景

多くの売場は「価格」でしか勝負していない。
しかし消費者は本来「意味」で購入している。

👉 価格競争から脱却する新しいアプローチ

---

## 🧩 機能

・POPコピー自動生成
・体験数字の提示
・非価格購買の促進

---

## 🎯 コンセプト

価格は支払数字、数値は体験数字。

---

## 👤 プロジェクト

MetaZero / メタPOP構想
